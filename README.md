# Pico SSD1306 OLED Library

Минималистичная и универсальная библиотека на Си для работы с OLED-дисплеями на базе чипа **SSD1306** (разрешение 128x64) через интерфейс I2C на микроконтроллере **Raspberry Pi Pico**. Также стоит сказать что это учебный проект и в будущем я добавлю больше. Также я приветствую подходящую критику и советы, так как я только начинаю.

## Как использовать

1. Скопируйте файлы `ssd1306.c` и `ssd1306.h` в папку вашего проекта (например, в `library/`).
2. Подключите библиотеку в вашем файле `CMakeLists.txt`.

### Пример кода (main.c)

```c
#include "pico/stdlib.h"
#include "ssd1306.h"

// Создаем объект дисплея
ssd1306_t display;

int main() {
    stdio_init_all();

    // Инициализация: объект, порт I2C, адрес (0x3C), пин SDA, пин SCL
    oled_init(&display, i2c0, 0x3C, 4, 5);

    // Рисуем точку по центру экрана (X: 64, Y: 32)
    ssd1306_draw_pixel(&display, 64, 32);

    // Отправляем буфер на реальный экран
    ssd1306_show(&display);

    while (true) {
        sleep_ms(1000);
    }
}
